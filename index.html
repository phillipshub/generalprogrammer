Open Source Software Projects

UML Diagram Maker: UML diagrams are important to programmers because they provide a clear, visual representation of a system’s structure and behavior, making it easier to design, communicate, and maintain complex software. They help identify relationships between components, clarify requirements, and support collaboration across teams during development. UML Diagram Maker transforms the daunting task of creating UML diagrams into a simple data entry process. The program gathers input and outputs a properly formatted UML diagram that can easily be placed directly inside the programmer's documentation.
Access Project: UML Diagram Maker

Simple Benchmark Test: Monitoring server performance over time is essential for maintaining system reliability, optimizing resource usage, and ensuring a seamless user experience. By collecting and analyzing metrics such as server load. Simple Benchmark Test (SBT) is a light weight tool that runs a simple task on a schedule keeping records of the date and time required to run the program.
Access Project: Simple Benchmark Test

Prime Race: Famous retired Microsoft Engineer David Plumber held a competition between programming languages to solve the age old dispute as to what programming language is the fastest. My submission is in C++ a compiled programming language. The given task was to see how many times your program could correctly find all prime numbers in a given range within a certain time span. I did not win the competition but did however receive encouraging feedback complementing how clean my code is.
Access Project: Prime Race
